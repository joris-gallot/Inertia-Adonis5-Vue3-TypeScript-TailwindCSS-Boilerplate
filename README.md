# Inertia-Adonis5-Vue3-TypeScript-TailwindCSS-Boilerplate

## Installation

Install NPM dependencies :

```bash
npm ci
```

### Compile Assets

Build assets :

```bash
node ace mix:build
```

Build assets and watch for file changes :

```bash
node ace mix:watch
```

Build assets for production :

```bash
node ace mix:build --production
```

### Run server

Run adonis server :

```bash
npm run dev
```
